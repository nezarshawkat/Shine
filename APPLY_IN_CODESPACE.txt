SHINE CUMULATIVE UPDATE

1. Upload this ZIP to the root of the Shine repository in GitHub Codespaces.
2. Extract it at the repository root and allow all included files to overwrite.
3. Do not upload the private local env.txt file to GitHub.
4. In Render environment variables, set:
   AUTO_ACTIVITY_ENABLED=true
   AUTO_ACTIVITY_COMMUNITY_POST_RATE=0.25
5. Run:
   cd backend
   npm install
   npm test
6. Commit the extracted files, then deploy the frontend and backend normally.

Included behavior:
- personalized keyword-based feed with unseen posts first
- local SQLite feed interaction tracking
- Shine community maintained at 2,234 members, including all 50 seeded profiles
- about 25% of future AI posts assigned to Shine
- community destinations restored in every post-create page through user memberships
- white, clearly outlined community-card keywords in dark mode

The private local SQLite data itself is not included in source control. Membership
seeding runs automatically when the backend starts.
